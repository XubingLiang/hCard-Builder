# hCard Builder
Prerequisite:
- NodeJS(latest)


## Set up

update npm
```sh
$ npm install npm@latest -g
```

install dependencies for react
```sh
$ cd hcard-builder
$ yarn install
```

run the project
```sh
$ yarn start
```

## Guide

**Time Spent:** around 4.5 hours
**Use UI library:** react-bootstrap
